Calorie Tracker

Project Github Repo

This project was built using the backbone.js framework. Udacity challenged students to learn Backbone by themselves from resources available on the web. The idea was then for them to build an app that allowed users to search food from an API and add them to a sum table. I also ended up implementing the calendar as I felt it made sense. Originally the data was stored locally using the localStorage object, but currently the data gets saved in a server using Firebase.

Downtown Map

Project Github Repo

The final project of a Udacity course, it's main purpose was for students to practice using an organizational framework, in this case, Knockout.js, and AJAX requests. It consists of a selection of bars and clubs in Porto's downtown. The app fetches the locations from a JSON and using Google Maps' API, populates the markers. The infowindows show information fetched form Foursquare, that include rating, address and closing time. There is also a live search box that filters the list.

Arcade Game (Frogger clone)

Interactive Resume

Project Github Repo
