mrMDB

View Github Repo

A personal project, built using Angular 1.x, SASS and Ecmascript6 (2015/Harmony), this project allows users to search for a movie and get the ratings of the main film review websites, IMDb, Rotten Tomatoes and Metacritic. It also includes general info about the movie. It was built in part, as a way to practice some new stuff, like Angular 1.x, ES6, SASS, PostCSS (autoprefixer), modular JS (import/export es6 syntax), Webpack as a bundler and replacement for Gulp, etc. Most of the data comes from omdb, a non-official imdb api.

Calorie Tracker

View Github Repo

This project, built using Backbone.js, was created following a challenge by Udacity. The ideia was to learn Backbone.js from resources around the web and build an app that allowed users to look for food items from a DB and add them to a list that displayed the total calories. I ended up implementing the calendar as I felt it made perfect sense. For demonstration purposes, the data gets stored in a Firebase server.

Downtown Map

View Github Repo

Part of Udacity's "Javascript Design Patterns" course, this project required students to build an app from scratch to practice using AJAX requests and an organizational framework, in this case, Knockout.js. It consists of a selection of bars and clubs in Porto's downtown. By providing the locations, the app uses Google Maps' API to populate the map with the markers. The infowindows show information fetched form Foursquare, that include rating, address, closing time and popular comments. There is also a live search box and a few buttons to help filter the list.
